PGN to FASTA | Chess to DNA

Internal pipeline: PGN to UCI to FASTA. Paste a PGN file with one or more games. Each game becomes an independent FASTA sequence. Each half-move yields two codons: from-square + to-square. Promotions are signaled by impossible extra destination–destination windows, without breaking reading in codon pairs.

Input

Reference board

The mapping is bijective: 64 squares to 64 codons.
